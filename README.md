# MemAdapter Project Page

This repository hosts a static academic project page for:

**MemAdapter: Structuring Retrieved Memories for VLM-Based Embodied Planning**

## Deploy on GitHub Pages

1. Open your repository on GitHub.
2. Go to **Settings → Pages**.
3. In **Build and deployment**, choose:
   - **Source**: Deploy from branch
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Save, then wait for GitHub Pages to publish the site.

## File placement guide

- Main page: `index.html`
- Styles: `style.css`
- Paper and downloadable materials:
  - `assets/paper.pdf`
  - `assets/poster.pdf`
  - `assets/supplementary.pdf`
- Figures and teaser images:
  - `assets/teaser.png`
  - `assets/framework.png`
  - `images/method_overview.png`
  - `images/qualitative_results.png`
- Videos:
  - `videos/overview.mp4`
  - `videos/success_case_1.mp4`
  - `videos/replanning_case.mp4`
  - `videos/failure_case.mp4`

Replace placeholder files with your final assets while keeping the same file names (or update the paths in `index.html`).

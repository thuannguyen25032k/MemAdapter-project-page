Abstract

Memory-augmented VLM planning is promising for long-horizon embodied tasks, but directly injecting raw retrieved memories into a planner can be unreliable because such memories are verbose, heterogeneous, and unstructured. MemAdapter addresses this issue by converting retrieved spatial, temporal, episodic, and semantic memories into structured task-level guidance: a foresight plan, feasibility criteria, and a fallback strategy. This guidance helps the planner sequence long-horizon tasks, allows the critic to verify action feasibility, and supports recovery from failures under partial observability.

Method Overview

1. Retrieve Multi-type Memories

Collect spatial, temporal, episodic, and semantic memories relevant to the current instruction.

2. Adapt Memories into Structured Guidance

Generate a foresight plan, feasibility criteria, and a fallback strategy from heterogeneous retrieved memories.

3. Plan, Verify, Execute, and Update Memory

Use guidance for planning and critic verification, execute actions, and update memory for the next decision step.

Qualitative Examples

  • Object search under partial observability: memory-guided planning narrows plausible search trajectories.
  • Feasibility checking before execution: critic criteria prevent low-probability or infeasible action commitments.
  • Recovery after invalid or failed actions: fallback strategy supports robust replanning and task continuation.
